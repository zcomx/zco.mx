Marking a book complete means a CBZ file will be created and shared with others via two file sharing networks: bittorrent and direct connect.

Before proceeding:

  • Ensure all images have been uploaded and are in the correct order.
  • If you want to use a custom indicia image, make sure it has been uploaded before marking the book complete.
Marking a book as complete has the following effect:
  • A CBZ file is created
  • The file is added to the Direct Connect network
  • A torrent is created for the file and the torrent is seeded
  • The file is added to the torrent containing all books from the cartoonist
  • The file is added to the torrent containing all books from zco.mx
  • The file pages are locked from further modifications

The process can take several minutes.
