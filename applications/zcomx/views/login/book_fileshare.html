Releasing a book for filesharing means a CBZ file will be created and shared with others via two file sharing networks: bittorrent and direct connect.

Before proceeding:

  • Ensure the book is set as completed.
  • If you want to use a custom indicia image, make sure it has been uploaded before releasing the book for filesharing.
Releasing a book for filesharing has the following effect:
  • A CBZ file is created
  • The file is added to the Direct Connect network
  • A torrent is created for the file and the torrent is seeded
  • The file is added to the torrent containing all books from the cartoonist
  • The file is added to the torrent containing all books from zco.mx
  • The book is locked from further modifications

The process can take up to an hour if many images need to be resized to fit the CBZ format.
